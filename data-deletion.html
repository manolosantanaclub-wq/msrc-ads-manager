MSRC Ads Manager · Internal Tool

User Data Deletion Instructions

Effective date: 23 March 2026  ·  Last updated: 23 March 2026

Summary: MSRC Ads Manager does not store personal data from any individual. No data deletion request is required. However, this page explains how to revoke the application's access to your Meta account.

1. About data stored by this application

MSRC Ads Manager is a private internal tool operated exclusively by Racquets Club Marbella S.L. It does not create accounts for any external user, does not store personal data of any individual, and does not retain Meta API data on any external server.

2. How to revoke access

If you are the authorised operator and wish to revoke this application's access to your Meta account:

  1. Go to your Facebook account settings at facebook.com/settings.
  2. Navigate to Security and Login → Apps and Websites.
  3. Find MSRC Ads Manager in the list of active apps.
  4. Click Remove to revoke all permissions.
  5. Confirm the removal when prompted.

3. How to delete your Meta advertising data

Any advertising data stored within Meta's platforms can be managed directly through Meta Business Manager. For further information, refer to Meta's Privacy Policy.
